# APKDominos

Repositório de aprendizagem de C# e .NET para developers com background em Python e JavaScript.

## Conteúdo

- 📖 [**LEARNING_ROADMAP.md**](./LEARNING_ROADMAP.md) — Roteiro prático com mapeamento de conceitos Python/JS → C#, sintaxe essencial, LINQ, Async/Await e estrutura de Web API.
- 🚀 [**HelloWorldAPI/**](./HelloWorldAPI/) — Mini-projeto de Web API em .NET para entender o fluxo de dados end-to-end.

## Início Rápido

```bash
# Pré-requisito: .NET SDK 8.0+  https://dotnet.microsoft.com/download

cd HelloWorldAPI
dotnet run
```

Acede a `http://localhost:5000/swagger` para explorar os endpoints via Swagger UI.
